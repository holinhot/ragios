Server Setup

Ragios server requires CouchDB, to install CouchDB on Ubuntu

sudo aptitude install couchdb

Ragios runs as a CouchDB admin user, first create a CouchDB admin user, you can do this from the terminal.
Enter ruby’s irb (interactive ruby shell) and type:

 require 'leanback'
 Couchdb.set_config :section => "admins", :key => "ragios_server", :value => "ragios"

Note: The above example above assumes you are using the default values, the admin user’s username is ‘ragios_server’ and ‘password’ is ‘ragios’ you should change these values to the username and password you want for the CouchDB admin. This can also be done by going to Futon (CouchDB’s web based admin panel at http://localhost:5984/_utils/)

Now from the Ragios root directory, open the file config.rb.

Add the username and password of the CouchDB admin you created;

database_admin = {username: 'ragios_server',
                 password: 'ragios'}

Replace the quoted strings with the username and password respectively. Or leave as is if you are using default values.
